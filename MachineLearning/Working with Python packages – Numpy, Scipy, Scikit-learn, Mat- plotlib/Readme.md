## Experiment 1: Working with Python packages
### Objective
To explore Numpy, Pandas, Scipy, Scikit-learn, and Matplotlib by performing Exploratory Data Analysis (EDA) and preprocessing on various datasets.

### Dataset: Loan Amount Prediction
- **Task:** Classification (Supervised Learning)
- **Goal:** Predict if a loan will be approved based on applicant details.

### Libraries Used
- **Numpy**: Array manipulation and mathematical operations.
- **Pandas**: Data handling and preprocessing.
- **Matplotlib/Seaborn**: Data visualization (Histograms, Box plots).
- **Scikit-learn**: Basic model training and evaluation.

### How to Run
1. Clone the repository.
2. Install dependencies: `pip install -r requirements.txt`.
3. Open `Experiment-1/Loan_Prediction.ipynb` in Jupyter Notebook.
